Формулы кинематики с пояснениями по физике



Кинематика — раздел физики, занимающийся исследованием законов движения идеальных тел.

Основные формулы с пояснениями, которые помогут в решении заданий ЕГЭ по физике: движение, скорость, ускорение.

 

Путь, время, скорость

S=v∗tS=v *t

  • S - путь
  • v - скорость
  • t - время


Равномерное движение

x=x0+v∗tx=x_0 + v*t

  • x - координата
  • x0 - начальная координата
  • v - скорость
  • t - время

Равномерно ускоренное движение: ускорение

a=v−v0ta=\frac { v - v_0 } { t }

  • a - ускорение
  • v - скорость
  • v0 - начальная скорость
  • t - время

Равномерно ускоренное движение: скорость

v=v0+atv=v_0 + at

  • v - скорость
  • v0 - начальная скорость
  • a - ускорение
  • t - время

Равномерно ускоренное движение: путь

S=vt+at22S=vt + \frac { at^2 } { 2 }

  • s - путь
  • v - скорость
  • t - время
  • a - ускорение

Равномерно ускоренное движение: координата

x=x0+vt+at22x=x_0 + vt + \frac { at^2 } { 2 }

  • x - координата
  • x0 - начальная координата
  • v - скорость
  • t - время
  • a - ускорение


Высота тела, брошенного вертикально вверх (вниз)

h=h0+v0t−gt22h=h_0 + v_ { 0 } t - \frac { gt^2 } { 2 }

  • h - высота
  • h0 - начальная высота
  • v0 - начальная скорость
  • t - время
  • g - ускорение свободного падения


Скорость тела, брошенного вертикально вверх (вниз)

v=v0−gtv=v_0 - gt

  • v - скорость
  • v0 - начальная скорость
  • g - ускорение свободного падения
  • t - время


Скорость, ускорение, время

v=atv=at

  • v - скорость
  • g - ускорение свободного падения
  • t - время


Центростремительное ускорение

a=v2Ra=\frac { v^2 } { R }

  • a - центростремительное ускорение
  • v - скорость
  • R - радиус


Угловая скорость

ω=ϕt\omega=\frac { \phi } { t }

  • ω - угловая скорость
  • φ - угол
  • t - время


Равномерное круговое движение

l=Rϕl=R\phi

  • l - длина дуги окружности
  • R - радиус
  • φ - угол

Равномерное круговое движение: линейная скорость

v=Rωv=R \omega

  • v - линейная скорость
  • R - радиус
  • ω - угловая скорость

T=tNT=\frac { t } { N }

  • T - период
  • t - время
  • N - число вращений

Частота вращения
